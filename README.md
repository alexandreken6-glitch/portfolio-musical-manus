## Sobre o Artista

**Lúmen** é um projeto de trap emocional criado por **Alexandre Marques** (Xmigi), artista brasileiro com alma japonesa. O projeto nasce da interseção entre duas culturas — a saudade brasileira e o wabi-sabi japonês — transformando experiências de ansiedade, fé, perda e amor em composições que tocam onde as palavras comuns não chegam.

O nome *Lúmen* vem do latim: a menor unidade de luz. Cada música é um fragmento de luz no escuro.

## Identidade Visual

O projeto Lúmen usa uma estética de **tinta japonesa + glow digital** — caligrafia sumi-e encontrando neon. As cores principais são:

- **Preto profundo** `#0a0a0a` — o silêncio antes do som
- **Rosa neon** `#ff2d78` — a emoção crua
- **Dourado** `#f5c842` — a fé e a esperança
- **Índigo** `#4a3f8f` — a introspecção
